# AI Image Direct Prompt v5

Product screenshot presentation mode.

Changes from v4:
- Feature screens are background-only decoration.
- Explicitly forbids any standalone illustration object in features: no buddy, mascot, character, robot, campervan, vehicle, cards, stickers, floating panels, or foreground objects.
- The phone + real screenshot must be the only concrete object.
- productVisualAnchor influences background mood only.

Reason:
- v4 still allowed models to add characters/cards over screenshots. v5 trades richness for preserving the real screenshot and consistent phone presentation.
