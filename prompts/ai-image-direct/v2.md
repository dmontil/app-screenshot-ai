# AI Image Direct Prompt v2

Screenshot-preservation iteration.

Changes from v1:
- Do not pass generated cover/previous generated images into feature generations.
- Feature generations receive only style reference + their real screenshot.
- Keep consistency through prompt wording rather than generated-image context.
- Cover asks for fewer objects to avoid contaminating later screens.

Hypothesis:
- Less image context should improve real screenshot fidelity and reduce copied campervan/card overlays.
- Consistency may be slightly weaker than v1, but phone/UI preservation should improve.
