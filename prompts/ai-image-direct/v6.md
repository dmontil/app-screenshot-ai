# AI Image Direct Prompt v6

Premium locked campaign system.

Changes from v5:
- Keeps feature screens phone-first and screenshot-preserving.
- Forces all feature screens into a consistent dark forest-green / warm cream / soft orange campaign system.
- Uses the same layout across features: headline top, large centered phone, subtle route/map background only.
- Forbids standalone illustrations in features: no buddy, mascot, campervan, cards, stickers, panels, or foreground objects.
- Tightens the cover into a mature editorial style: one campervan/route symbol, no cute face unless the reference is character-led.
- Explicitly forbids fake UI/text objects on the cover: no route cards, chips, speech bubbles, “AI” icon, badges, labels, numbers, or itinerary panels.

Reason:
- v5 produced strong screenshot preservation, especially screen 02, but the set still lacked top-tier campaign consistency: cover had fake text/AI-chip artifacts and screen 03 drifted to a lighter, softer system.
- v6 trades some playful richness for a more premium, coherent App Store set.

Success criteria:
- Real app UI remains readable and unmodified.
- Feature screens feel like one designed campaign, not separate generations.
- Cover looks premium and specific without fake readable content.
- Background decoration supports the route-planning concept without competing with the phone.
