# AI Image Direct Prompt v4

Feature minimal / buddy conditional iteration.

Changes from v3:
- Features stay screenshot-first.
- Buddy/mascot/character is allowed only if the style reference clearly has one.
- If the reference has no buddy, feature screens use only abstract background, gradients, subtle route lines, or tiny pins outside the phone.
- No campervan/robot/cards/map overlays on feature screens.
- Cover can still be expressive, but buddy is conditional on the reference style.

Hypothesis:
- Should improve screenshot preservation and reduce UI contamination while keeping enough campaign style.
